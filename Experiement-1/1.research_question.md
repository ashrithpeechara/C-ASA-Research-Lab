<h1 text-align="center"> Design, Configuration, and Performance Evaluation of a Traditional Enterprise Network </h1>



# Research Question 

How does a traditional manually configured enterprise network operate, and what are its baseline performance characteristics before introducing autonomic or cognitive networking concepts?

## Objectives
By the end of this experiment, you should be able to:

Design a basic enterprise network.
Configure routers and switches manually.
Assign IPv4 addresses.
Configure static routing.
Verify end-to-end connectivity.
Observe packet forwarding.
Capture network traffic.
Establish baseline performance metrics for future experiments.

## Learning Outcomes

After completing this experiment, you should understand:

Enterprise network architecture
Layer 2 vs Layer 3
IP Addressing
Static Routing
ARP
ICMP
Ethernet Switching
Packet forwarding
Router forwarding decisions
