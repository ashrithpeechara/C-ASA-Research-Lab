# Experiment 01

## Title

Traditional Network Architecture using Mininet

## Objective

To understand the architecture and operation of a traditional Layer-2 network using Mininet and Open vSwitch.

## Tasks

1. Verify the lab environment.
2. Create a Mininet topology with one switch and three hosts.
3. Display the network topology.
4. Display Open vSwitch configuration.
5. Verify host connectivity.
6. Observe MAC address learning.
7. Capture packets using tcpdump.
8. Analyze ARP and ICMP communication.

## Expected Learning Outcomes

- Understand Layer-2 switching
- Understand MAC learning
- Understand ARP
- Understand ICMP
- Understand Open vSwitch in standalone mode
